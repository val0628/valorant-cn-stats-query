# 隐私说明

WGame VAL Query 只用于展示玩家公开游戏资料和公开战绩信息。

## 项目展示的信息

可能展示：

- 游戏昵称和 ID
- 近期战绩
- 当前房间及当前对局玩家的公开游戏资料
- 英雄
- KDA
- ACS
- 地图
- 段位和 RR 变化
- 官方公开 IP 属地

## 项目不会展示的信息

不会展示：

- 玩家真实 IP
- 玩家真实姓名
- QQ 号
- Cookie
- token
- 私人聊天内容
- 本地抓包文件

## 实时账号选择

网吧内多名玩家可能共享同一个公网 IP。实时页面只使用当前网络信息缩小有效账号候选范围，并要求用户按游戏昵称选择自己的账号。

- 不会因为公网 IP 相同就自动加载最新对局。
- 未选择账号时不会自动进入房间或当前对局。
- 公共页面不返回内部 session、subject、PartyID、原始事件 ID 或 token 来源。
- 账号选择只在当前浏览器中临时保存。

## Issue 注意事项

提交反馈时，请不要上传：

- Cookie
- token
- 抓包文件
- 登录二维码
- QQ 号
- 任何能证明你或他人真实身份的信息
