# 隐私说明

WGame VAL Query 只用于展示玩家公开游戏资料和公开战绩信息。

## 项目展示的信息

可能展示：

- 游戏昵称和 ID
- 近期战绩
- 英雄
- KDA
- ACS
- 地图
- 段位和 RR 变化
- 官方公开 IP 属地

## 项目不会展示的信息

不会展示：

- 玩家真实 IP
- 玩家真实姓名
- QQ 号
- Cookie
- token
- 私人聊天内容
- 本地抓包文件

## Issue 注意事项

提交反馈时，请不要上传：

- Cookie
- token
- 抓包文件
- 登录二维码
- QQ 号
- 任何能证明你或他人真实身份的信息
