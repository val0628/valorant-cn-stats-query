# 隐私说明

WGame VAL Query 只用于展示玩家公开游戏资料和公开战绩信息。

## 项目展示的信息

可能展示：

- 游戏昵称和 ID
- 近期战绩
- 当前房间及当前对局玩家的公开游戏资料
- 英雄
- KDA
- ACS
- 地图
- 段位和 RR 变化
- 官方公开 IP 属地

## 项目不会展示的信息

不会展示：

- 玩家真实 IP
- 玩家真实姓名
- QQ 号
- Cookie
- token
- 私人聊天内容
- 本地抓包文件

## 实时账号选择

网吧内多名玩家可能共享同一个公网 IP。实时页面只使用当前网络信息缩小有效账号候选范围，并要求用户按游戏昵称选择自己的账号。

- 不会因为公网 IP 相同就自动加载最新对局。
- 未选择账号时不会自动进入房间或当前对局。
- 公共页面不返回内部 session、subject、PartyID、原始事件 ID 或 token 来源。
- 账号选择只在当前浏览器中临时保存。

## 实时功能风险确认记录

实时对局会处理与游戏实时状态有关的网络请求，用户必须先阅读并逐项确认[风险知情与使用协议](../DISCLAIMER.md)。为证明风险提示和主动确认过程，服务端保存：

- 协议版本和协议正文 SHA-256 摘要
- 确认时间和随机确认编号
- 经哈希处理的网络和浏览器环境摘要

确认记录不保存原始 IP 地址。未撤回的确认记录最长保存 365 天。浏览器确认凭证有效期为 180 天；协议或处理方式发生重要变化时会要求重新确认。

已确认用户可以在实时页面点击“一键撤回并永久锁定”。撤回不会删除原确认记录，而会追加撤回时间；当前浏览器标识的哈希摘要及撤回记录会在实时功能持续提供期间保留，用于执行不可恢复锁定。本站没有账号体系，用户清除全部站点数据后可能无法继续识别同一浏览器。

## Issue 注意事项

提交反馈时，请不要上传：

- Cookie
- token
- 抓包文件
- 登录二维码
- QQ 号
- 任何能证明你或他人真实身份的信息
