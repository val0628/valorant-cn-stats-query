# 隐私说明

WGame VAL Query 用于展示玩家公开游戏资料和公开战绩信息，并在必要范围内记录功能访问情况，以提供服务、保障安全、排查滥用、定位故障和统计使用情况。

在线说明页：[https://val.skzspro.cn/privacy](https://val.skzspro.cn/privacy)

## 会记录的使用数据

### 公共查询

用户提交昵称查询时，服务端保存：

- 查询时间和访问 IP；
- 用户提交的昵称#ID；
- 解析到的游戏昵称和账号标识；
- 查询成功、未找到或上游异常等处理结果。

### 实时对局

用户选择实时账号、查看房间近 3 场或查看当前对局时，服务端保存：

- 使用时间和访问 IP；
- 游戏会话来源 IP；
- 用户主动选择的游戏昵称和账号标识；
- 使用的实时功能、访问入口和处理结果。

上述公共查询和实时功能使用记录默认保存 180 天，期限届满后自动删除。数据库和管理页面仅在运行服务的本机控制中心开放，不通过公共页面或公开接口提供。

## 项目展示的信息

可能展示：

- 游戏昵称和 ID
- 近期战绩
- 当前房间及当前对局玩家的公开游戏资料
- 英雄
- KDA
- ACS
- 地图
- 段位和 RR 变化
- 官方公开 IP 属地

## 项目不会展示的信息

不会展示：

- 玩家真实 IP
- 玩家真实姓名
- QQ 号
- Cookie
- token
- 私人聊天内容
- 本地抓包文件

“不会展示玩家真实 IP”是指不会把真实 IP 放在公共查询结果、实时对局结果或其他公开页面中；服务端仍会按本说明记录访问 IP 和实时会话来源 IP。

## 实时账号选择

网吧内多名玩家可能共享同一个公网 IP。实时页面只使用当前网络信息缩小有效账号候选范围，并要求用户按游戏昵称选择自己的账号。

- 不会因为公网 IP 相同就自动加载最新对局。
- 未选择账号时不会自动进入房间或当前对局。
- 公共页面不返回内部 session、subject、PartyID、原始事件 ID 或 token 来源。
- 账号选择只在当前浏览器中临时保存。

## 实时功能风险确认记录

实时对局会处理与游戏实时状态有关的网络请求，用户必须先阅读并逐项确认[风险知情与使用协议](../DISCLAIMER.md)。为证明风险提示和主动确认过程，服务端保存：

- 协议版本和协议正文 SHA-256 摘要
- 确认时间和随机确认编号
- 经哈希处理的网络和浏览器环境摘要

确认记录不保存原始 IP 地址。未撤回的确认记录最长保存 365 天。浏览器确认凭证有效期为 180 天；协议或处理方式发生重要变化时会要求重新确认。

风险确认记录与上文的功能使用记录相互独立：确认记录不保存原始 IP，不代表公共查询和实时功能使用审计不保存 IP。

已确认用户可以在实时页面点击“一键撤回并永久锁定”。撤回不会删除原确认记录，而会追加撤回时间；当前浏览器标识的哈希摘要及撤回记录会在实时功能持续提供期间保留，用于执行不可恢复锁定。本站没有账号体系，用户清除全部站点数据后可能无法继续识别同一浏览器。

## 查询、更正与删除

如需依法查询、更正或删除相关记录，可通过项目 [GitHub Issues](https://github.com/val0628/valorant-cn-stats-query/issues) 联系维护者。为定位记录，维护者可能需要请求者在非公开沟通中提供大致访问时间、查询昵称或 IP 等必要线索；请勿在公开 Issue 中直接发布敏感信息。

## Issue 注意事项

提交反馈时，请不要上传：

- Cookie
- token
- 抓包文件
- 登录二维码
- QQ 号
- 任何能证明你或他人真实身份的信息
