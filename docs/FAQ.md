# FAQ

## 这个项目能自己部署吗？

当前公开仓库不提供完整服务端部署代码。

项目目标是让用户直接访问主站使用：

```text
https://val.skzspro.cn/public
```

## 为什么不公开核心代码？

核心服务涉及生产环境稳定性、登录态管理、抓包链路和风控适配。为了保护服务可用性和避免误用，暂时不公开核心服务端。

公开仓库主要用于：

- 项目介绍
- 使用说明
- 截图展示
- 收集反馈
- 发布更新记录

## 会显示玩家真实 IP 吗？

不会。

项目只展示官方接口返回的公开 IP 属地，例如 `贵州`、`四川`、`北京`。不会提取、保存或展示玩家真实 IP。

## 查询失败怎么办？

可以在 Issue 里提交：

- 昵称#ID
- 查询时间
- 页面提示
- 你希望查询的场次

不要提交 Cookie、token 或抓包文件。

## QQ 群插件可以用吗？

QQ 群文本查询已经在项目链路中支持。公开接入方式会后续整理，如果你有 QQ 群接入需求，可以通过 Issue 说明使用场景。

## 这个项目和 5EBOX 是什么关系？

没有从属关系。

只是功能体验上都属于“玩家开局前/对局中快速看信息”的工具。这个项目面向国服无畏契约和 WeGame 生态重新开发。
